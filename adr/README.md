# ADRs

Este diretório reúne registros de decisão arquitetural da KdevX Engineering Platform.

ADRs registram decisões relevantes, seu contexto e suas consequências.

## Convenção

- Usar o prefixo `ADR-`.
- Usar numeração sequencial.
- Escrever em português do Brasil.
- Registrar decisões que afetem arquitetura, governança, IA, padrões, produtos ou propriedade intelectual.

## ADR inicial

- `ADR-0000_FOUNDATION.md`: decisão de criar a KEP como ativo estratégico proprietário.
