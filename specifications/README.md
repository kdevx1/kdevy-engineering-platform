# Especificações

Este diretório reúne as especificações da KDEVY Engineering Platform.

As especificações pertencem à KDEVY e devem orientar produtos, automações, agentes de IA e decisões técnicas.

## Convenção

- Usar o prefixo `KES-`.
- Registrar um identificador numérico.
- Escrever títulos claros.
- Manter linguagem em português do Brasil.

## Documento inicial

- `KES-0000_MASTER_SPECIFICATION.md`: especificação-mestre inicial da KEP.
