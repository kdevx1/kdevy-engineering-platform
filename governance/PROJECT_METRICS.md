# Métricas do Projeto

## Propósito

Este documento define métricas iniciais para acompanhar a evolução da KdevX Engineering Platform.

As métricas devem ajudar pessoas e agentes de IA a entender maturidade, cobertura e qualidade operacional da plataforma.

## Métricas iniciais

| Métrica | Descrição | Estado inicial |
| --- | --- | --- |
| Cobertura de documentação | Percentual de áreas da KEP com README ou documento orientador | Base criada |
| ADRs registradas | Quantidade de decisões arquiteturais documentadas | 1 |
| Políticas registradas | Quantidade de políticas formais | 2 |
| Especificações registradas | Quantidade de especificações KES | 1 |
| Produtos consumidores | Quantidade de produtos que consomem a KEP | 1 planejado |
| Maturidade de IA | Grau de formalização do uso de IA na engenharia | Inicial |

## Cadência de revisão

As métricas devem ser revisadas a cada onda do roadmap ou sempre que houver mudança relevante de governança.
