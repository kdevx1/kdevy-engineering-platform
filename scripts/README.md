# Scripts

Este diretório reunirá scripts e automações da KDEVY Engineering Platform.

## Diretrizes

- Scripts devem ter propósito claro.
- Scripts devem evitar dependências desnecessárias.
- Scripts que alterem estado devem documentar impacto e modo de reversão.
- Scripts não devem copiar código-fonte do AutoBody ERP.
- Scripts devem respeitar políticas de classificação e propriedade intelectual.

## Estado atual

Nenhum script operacional foi criado na versão 0.1.0 Foundation.
