# KIPs

Este diretório reunirá propostas de melhoria da KDEVY.

KIPs são propostas estruturadas de melhoria para a KDEVY Engineering Platform.

## Quando criar uma KIP

- Para mudanças relevantes na estrutura da KEP.
- Para novos módulos, padrões ou processos.
- Para melhorias que afetem múltiplas áreas.
- Para propostas que precisem de discussão antes de virar decisão.

## Relação com ADR

Uma KIP pode evoluir para uma ADR quando houver decisão arquitetural ou estratégica formal.
